üéÆ Install Panel Pterodactyl

Tutorial lengkap instalasi Pterodactyl Panel untuk mengelola game server dengan interface web yang modern dan powerful. Panel ini mendukung berbagai jenis game server seperti Minecraft, FiveM, CS:GO, dan banyak lagi.

üöÄ Apa itu Pterodactyl Panel?

Pterodactyl adalah panel kontrol game server open-source yang dirancang untuk kemudahan penggunaan, keamanan, dan skalabilitas. Panel ini memungkinkan Anda mengelola multiple game server dari satu dashboard web yang intuitif.

üìã System Requirements

Sebelum memulai instalasi, pastikan VPS Anda memenuhi requirements minimum berikut:

Minimum Requirements

OS
Ubuntu 20.04+ / CentOS 7+
RAM
1GB (2GB recommended)
Storage
10GB+ SSD
PHP
8.0+ (8.1 recommended)
‚ö†Ô∏è Important Notes
  • Tutorial ini menggunakan Ubuntu 22.04 LTS
  • Pastikan Anda memiliki akses root atau sudo privileges
  • Domain/subdomain sudah dikonfigurasi menuju IP server
  • Port 80 dan 443 harus terbuka untuk web traffic

üîß Install Dependencies

1Update System & Install Basic Packages

Pertama, update system dan install packages dasar yang dibutuhkan:

Terminal 
# Update system packages
sudo apt update && sudo apt upgrade -y

# Install basic dependencies
sudo apt install -y curl wget gnupg2 software-properties-common apt-transport-https ca-certificates lsb-release

2Install PHP 8.1

Install PHP dan ekstensi yang dibutuhkan oleh Pterodactyl:

Terminal 
# Add PHP repository
sudo add-apt-repository -y ppa:ondrej/php
sudo apt update

# Install PHP and required extensions
sudo apt install -y php8.1 php8.1-{cli,gd,mysql,pdo,mbstring,tokenizer,bcmath,xml,fpm,curl,zip}

3Install Composer

Composer diperlukan untuk mengelola dependencies PHP:

Terminal 
# Download dan install Composer
curl -sS https://getcomposer.org/installer | sudo php -- --install-dir=/usr/local/bin --filename=composer

# Verify installation
composer --version

4Install MariaDB

Install database server untuk menyimpan data panel:

Terminal 
# Install MariaDB
sudo apt install -y mariadb-server

# Secure MariaDB installation
sudo mysql_secure_installation
üí° MariaDB Security Setup

Saat menjalankan mysql_secure_installation, jawab:

  • Set root password: Y
  • Remove anonymous users: Y
  • Disallow root login remotely: Y
  • Remove test database: Y
  • Reload privilege tables: Y

5Install Redis

Redis digunakan untuk caching dan session management:

Terminal 
# Install Redis
sudo apt install -y redis-server

# Start dan enable Redis
sudo systemctl start redis-server
sudo systemctl enable redis-server

# Test Redis connection
redis-cli ping

üì• Download Pterodactyl Panel

6Create Directory & Download

Buat direktori untuk panel dan download source code:

Terminal 
# Create directory
sudo mkdir -p /var/www/pterodactyl
cd /var/www/pterodactyl

# Download latest release
sudo curl -Lo panel.tar.gz https://github.com/pterodactyl/panel/releases/latest/download/panel.tar.gz

# Extract archive
sudo tar -xzvf panel.tar.gz

# Remove archive
sudo rm panel.tar.gz

7Set Permissions

Set ownership dan permissions yang tepat:

Terminal 
# Set ownership to web server user
sudo chown -R www-data:www-data /var/www/pterodactyl/*

# Set directory permissions
sudo chmod -R 755 storage/* bootstrap/cache/

üóÑÔ∏è Database Setup

8Create Database

Buat database dan user untuk Pterodactyl Panel:

Terminal 
# Login ke MySQL
sudo mysql -u root -p

# Jalankan commands berikut di MySQL console:
CREATE USER 'pterodactyl'@'127.0.0.1' IDENTIFIED BY 'yourpassword';
CREATE DATABASE panel;
GRANT ALL PRIVILEGES ON panel.* TO 'pterodactyl'@'127.0.0.1' WITH GRANT OPTION;
FLUSH PRIVILEGES;
EXIT;
üîê Security Note

Ganti yourpassword dengan password yang kuat. Password ini akan digunakan di konfigurasi panel nanti.

üîß Environment Setup

12Configure Environment File

Edit file .env untuk mengatur konfigurasi database dan aplikasi:

Terminal 
# Edit environment file
sudo nano /var/www/pterodactyl/.env

Update konfigurasi berikut:

.env 
# Application
APP_URL=https://panel.yourdomain.com
APP_TIMEZONE=Asia/Jakarta

# Database
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=panel
DB_USERNAME=pterodactyl
DB_PASSWORD=yourpassword

# Cache & Sessions
CACHE_DRIVER=redis
SESSION_DRIVER=redis
QUEUE_CONNECTION=redis

# Redis
REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

# Mail Configuration (Optional)
MAIL_DRIVER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=your-email@gmail.com
MAIL_PASSWORD=your-app-password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=your-email@gmail.com
MAIL_FROM_NAME="Pterodactyl Panel"
üìß Mail Configuration

üåê Web Server Configuration

14Install Nginx

Install dan configure Nginx sebagai web server:

Terminal 
# Install Nginx
sudo apt install -y nginx

# Start dan enable Nginx
sudo systemctl start nginx
sudo systemctl enable nginx

15Create Nginx Virtual Host

Buat konfigurasi virtual host untuk panel:

Terminal 
# Create nginx config
sudo nano /etc/nginx/sites-available/pterodactyl.conf

Masukkan konfigurasi berikut:

pterodactyl.conf 
server {
    listen 80;
    server_name panel.yourdomain.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name panel.yourdomain.com;

    root /var/www/pterodactyl/public;
    index index.html index.htm index.php;
    charset utf-8;

    # SSL Configuration (akan dikonfigurasi dengan Certbot)
    # ssl_certificate /etc/letsencrypt/live/panel.yourdomain.com/fullchain.pem;
    # ssl_certificate_key /etc/letsencrypt/live/panel.yourdomain.com/privkey.pem;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location = /favicon.ico { access_log off; log_not_found off; }
    location = /robots.txt  { access_log off; log_not_found off; }

    access_log off;
    error_log  /var/log/nginx/pterodactyl.app-error.log error;

    # allow larger file uploads and longer script runtimes
    client_max_body_size 100m;
    client_body_timeout 120s;

    sendfile off;

    location ~ \.php$ {
        fastcgi_split_path_info ^(.+\.php)(/.+)$;
        fastcgi_pass unix:/var/run/php/php8.1-fpm.sock;
        fastcgi_index index.php;
        include fastcgi_params;
        fastcgi_param PHP_VALUE "upload_max_filesize = 100M \n post_max_size=100M";
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        fastcgi_param HTTP_PROXY "";
        fastcgi_intercept_errors off;
        fastcgi_buffer_size 16k;
        fastcgi_buffers 4 16k;
        fastcgi_connect_timeout 300;
        fastcgi_send_timeout 300;
        fastcgi_read_timeout 300;
    }

    location ~ /\.ht {
        deny all;
    }
}
üåê Domain Configuration

Pastikan untuk mengganti panel.yourdomain.com dengan domain/subdomain yang sesuai.

16Enable Site & Install SSL

Enable virtual host dan install SSL certificate:

Terminal 
# Enable site
sudo ln -s /etc/nginx/sites-available/pterodactyl.conf /etc/nginx/sites-enabled/pterodactyl.conf

# Remove default site
sudo rm /etc/nginx/sites-enabled/default

# Test nginx configuration
sudo nginx -t

# Install Certbot for SSL
sudo apt install -y certbot python3-certbot-nginx

# Get SSL certificate
sudo certbot --nginx -d panel.yourdomain.com

17Restart Services

Restart semua services yang diperlukan:

Terminal 
# Restart services
sudo systemctl restart nginx
sudo systemctl restart php8.1-fpm
sudo systemctl restart mariadb
sudo systemctl restart redis-server

üîÑ Queue Workers Setup

18Create Systemd Service

Buat service untuk menjalankan queue workers:

Terminal 
# Create systemd service file
sudo nano /etc/systemd/system/pteroq.service

Masukkan konfigurasi berikut:

pteroq.service 
[Unit]
Description=Pterodactyl Queue Worker
After=redis-server.service

[Service]
User=www-data
Group=www-data
Restart=always
ExecStart=/usr/bin/php /var/www/pterodactyl/artisan queue:work --queue=high,standard,low --sleep=3 --tries=3
StartLimitInterval=180
StartLimitBurst=30
RestartSec=5s

[Install]
WantedBy=multi-user.target
Terminal 
# Enable dan start service
sudo systemctl enable pteroq.service
sudo systemctl start pteroq.service

# Check status
sudo systemctl status pteroq.service

üë§ Create Admin User

20Create First Admin User

Buat user admin pertama untuk mengakses panel:

Terminal 
# Navigate to panel directory
cd /var/www/pterodactyl

# Create admin user
sudo -u www-data php artisan p:user:make

Ikuti prompts untuk mengisi:

  • Email: Email admin (untuk login)
  • Username: Username admin
  • First Name: Nama depan
  • Last Name: Nama belakang
  • Password: Password yang kuat
  • Admin: Pilih yes

üîç Troubleshooting

üö® Common Issues

Berikut adalah masalah umum dan solusinya:

Permission Issues

Jika mengalami masalah permissions:

Terminal 
# Reset permissions
sudo chown -R www-data:www-data /var/www/pterodactyl
sudo chmod -R 755 /var/www/pterodactyl
sudo chmod -R 755 storage/* bootstrap/cache/

Queue Worker Not Running

Jika queue worker tidak berjalan:

Terminal 
# Check queue service status
sudo systemctl status pteroq.service

# Restart queue service
sudo systemctl restart pteroq.service

# Check logs
sudo journalctl -u pteroq.service -f

Database Connection Issues

Jika tidak bisa connect ke database:

Terminal 
# Test database connection
mysql -u pterodactyl -p -h 127.0.0.1 panel

# Check MariaDB status
sudo systemctl status mariadb

# View MariaDB logs
sudo tail -f /var/log/mysql/error.log

Nginx Configuration Issues

Jika ada masalah dengan Nginx:

Terminal 
# Test nginx configuration
sudo nginx -t

# Check nginx logs
sudo tail -f /var/log/nginx/error.log

# Restart nginx
sudo systemctl restart nginx
